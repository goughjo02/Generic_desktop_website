## Sample Website that contains features including:
* flexbox
* colour gradients
* svg
* interactive interface
* stateful componenets

## Check it out
1. `git clone`
2. `cd`
3. `yarn install`
4. `yarn start`

### Test it out
1. In `App.js`, edit the `num_steps` property of te `HomePage` component to `3`.
2. Slide the slider.
3. Click the tabs.
